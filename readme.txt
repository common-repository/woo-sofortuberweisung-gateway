== Installation ==
<h4>Plugin Installation</h4>
1️⃣  Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2️⃣  Activate the plugin through the 'Plugins' screen in WordPress
3️⃣  Configure the Plugin (Woocommerce->Settings->Payments->Klarna / Sofortüberweisung)

<h4>Plugin Configuration</h4>
1️⃣  The first step is to get your Configuration key from <a href='https://www.klarna.com/' target='blank'>Klarna.com</a>.
Watch this video and follow the steps to get such a key.
[youtube https://www.youtube.com/watch?v=NgjtALKCLeg]
